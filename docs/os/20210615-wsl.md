启动 `wsl ubuntu` 出现如下错误：

```
参考的对象类型不支持尝试的操作。

[已退出进程，代码为 4294967295]
```

 

解决：

在管理员模式下运行如下命令，并重启电脑即可：

```
netsh winsock reset
```
