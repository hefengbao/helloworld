> Ubuntu 16.04
> https://laravel-china.org/docs/laravel/5.7/scheduling/2287


```
sudo crontab -e
```

添加如下内容：

```
* * * * * cd /var/www/data &&  php artisan schedule:run >> /dev/null 2>&1
```

保存并退出。

重启 `cron` 服务：

```
service cron restart
```

查看添加的定时任务：

```
sudo crontab -u root -l
```
